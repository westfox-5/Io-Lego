IO,LEGO - Readme file

Il progetto consiste nella realizzazione di un'applicazione che comanda un robot. 
Tramite applicazione si posizionano dei colori in una griglia e, con la pressione di un bottone, il robot comincia la ricerca nella matrice cartacea.
Terminata la ricerca e il controllo dell'esattezza del colore inserito nella griglia, il robot si posiziona nella cella iniziale, rivolto verso il basso.
Sono permesse ricerche consecutive, poichè il robot torna sempre alla posizione iniziale.
Terminato l'utilizzo, bisogna spegnere il robot premendo contemporaneamente il tasto centrale e tasto giù sull'EV3.

NOTA BENE: 
Il robot va sempre posizionato nella cella contrassegnata come cella iniziale, rivolto verso il basso.
Inoltre ricordarsi di preparare la griglia cartacea con i cartoncini colorati forniti prima di avviare il robot.
È necessario che il robot venga avviato nella corretta posizione iniziale, rivolto verso il basso, poichè il giroscopio si basa sul fatto che lo 0 (il valore rilevato all'avvio) sia la direzione verso il basso. 

APPLICAZIONE:
L'applicazione Android è stata realizzata con Android Studio 3.3 ed è supportata da tutti i dispositivi con versione minima 5.0 (API livello 21).

LEGO EV3:
Il robot è stato programmato in linguaggio Java, utilizzando il firmware LEJOS (http://www.lejos.org/).
